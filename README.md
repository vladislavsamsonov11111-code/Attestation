## 📝 Описание программы
**Expense Tracker** — это графическое приложение для отслеживания личных расходов. 
Программа позволяет вести учёт финансов, фильтровать записи по категориям и датам, 
а также подсчитывать суммы расходов за выбранный период.

### Основные возможности:
- ✅ Добавление расходов с указанием суммы, категории и даты
- ✅ Просмотр всех записей в удобной таблице
- ✅ Фильтрация по категории и диапазону дат
- ✅ Подсчёт общей суммы расходов за период
- ✅ Расчёт среднего чека
- ✅ Удаление отдельных записей или всех сразу
- ✅ Автоматическое сохранение данных в JSON-файл
- ✅ Валидация ввода (положительная сумма, корректная дата)
